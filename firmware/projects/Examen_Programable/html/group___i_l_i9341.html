Descripción detallada

TFT LCD RGB 240x320 ILI9341 driver.

This driver provide functions to configure and control a 240x320 pixels TFT color display connected to the ESP-EDU. It uses a SPI port and 3 GPIOs to communicate with the ILI9341 LCD driver chip.

Autor
Albano Peñalva
Nota
Hardware connections:
Display ESP-EDU
SDO/MISO GPIO_22
LED 3V3
SCK GPIO_20
SDI/MOSI GPIO_21
DC/RS GPIOx
RESET GPIOx
CS GPIOx
GND GND
VCC 3V3
